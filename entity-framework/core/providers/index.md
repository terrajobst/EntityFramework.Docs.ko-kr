# <a name="database-providers"></a><span data-ttu-id="370db-102">데이터베이스 공급자</span><span class="sxs-lookup"><span data-stu-id="370db-102">Database Providers</span></span>

<span data-ttu-id="370db-103">Entity Framework Core는 플러그 인 라이브러리 호출 데이터베이스 공급자를 통해 다양한 데이터베이스에 액세스할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="370db-103">Entity Framework Core can access many different databases through plug-in libraries called database providers.</span></span>

## <a name="current-providers"></a><span data-ttu-id="370db-104">현재 공급자</span><span class="sxs-lookup"><span data-stu-id="370db-104">Current providers</span></span>

> [!IMPORTANT]  
> <span data-ttu-id="370db-105">EF Core 공급자는 다양한 원본을 바탕으로 작성됩니다.</span><span class="sxs-lookup"><span data-stu-id="370db-105">EF Core providers are built by a variety of sources.</span></span> <span data-ttu-id="370db-106">일부 공급자는 [Entity Framework Core 프로젝트](https://github.com/aspnet/EntityFrameworkCore)의 일부로 유지되지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="370db-106">Not all providers are maintained as part of the [Entity Framework Core Project](https://github.com/aspnet/EntityFrameworkCore).</span></span> <span data-ttu-id="370db-107">공급자를 고려할 때는 품질, 라이선싱, 지원 등을 평가하여 요구 사항을 충족하는지 확인해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="370db-107">When considering a provider, be sure to evaluate quality, licensing, support, etc. to ensure they meet your requirements.</span></span> <span data-ttu-id="370db-108">또한 자세한 버전 호환성 정보는 각 공급자의 설명서를 검토해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="370db-108">Also make sure you review each provider's documentation for detailed version compatibility information.</span></span>

> <span data-ttu-id="370db-109">EF Core 공급자는 일반적으로 부 버전 간에는 작동하지만 주 버전에서는 작동하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="370db-109">EF Core providers typically work across minor versions, but not across major versions.</span></span> <span data-ttu-id="370db-110">예를 들어 EF Core 2.1에 대해 출시된 공급자는 EF Core 2.2에서 작동하지만 EF Core 3.0에서는 작동하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="370db-110">For example, a provider released for EF Core 2.1 should work with EF Core 2.2, but will not work with EF Core 3.0.</span></span>
