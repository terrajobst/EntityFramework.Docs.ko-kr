<span data-ttu-id="8e174-113">EF Core(Entity Framework Core)는 2016년 처음 릴리스된 EF6를 완전히 다시 작성한 버전입니다.</span><span class="sxs-lookup"><span data-stu-id="8e174-113">Entity Framework Core (EF Core) is a complete rewrite of EF6 that was first released in 2016.</span></span> <span data-ttu-id="8e174-114">Nuget 패키지로 제공되며 [Microsoft.EntityFrameworkCore](https://www.nuget.org/packages/Microsoft.EntityFrameworkCore/)가 기본입니다.</span><span class="sxs-lookup"><span data-stu-id="8e174-114">It ships in Nuget packages, the main one being [Microsoft.EntityFrameworkCore](https://www.nuget.org/packages/Microsoft.EntityFrameworkCore/).</span></span> <span data-ttu-id="8e174-115">EF Core는 .NET Core 또는 .NET Framework에서 실행할 수 있는 크로스 플랫폼 제품입니다.</span><span class="sxs-lookup"><span data-stu-id="8e174-115">EF Core is a cross-platform product that can run on .NET Core or .NET Framework.</span></span>
