# <a name="porting-from-ef6-to-ef-core"></a>EF6에서 EF Core로 이식

EF Core의 기본적인 변경 내용 때문에, 변경할 중요한 이유가 있는 경우에만 EF6 응용 프로그램을 EF Core로 이동하는 것이 좋습니다. EF6에서 EF Core로의 이동은 업그레이드가 아닌 이식으로 이해해야 합니다.
